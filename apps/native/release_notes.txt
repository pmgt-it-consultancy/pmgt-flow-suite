## What's New

- Bluetooth printer connection polling — detects disconnected printers within ~15 seconds
- Auto-reconnect with up to 3 retries when a printer drops connection
- Error alerts when manual reconnect fails (Retry/Dismiss options)
- Reconnect button added to Printer Settings screen for each configured printer
- Extended status indicators: Connected (green), Reconnecting (amber), Connection Failed (red)

## Notes

- Polling runs every 15 seconds and pauses when the app is backgrounded
- After 3 failed auto-reconnect attempts, the status shows "Connection Failed" — tap Reconnect to try again
- Status updates are visible in both the system status dropdown and Printer Settings
