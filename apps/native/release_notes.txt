## What's New

- Service type on all receipts — printed and shared receipts now show Dine-In or Takeout label on every item
- Refund preserves service type — replacement orders created from refunds retain each item's dine-in/takeout designation
- Kitchen ticket service type — dine-in order screen now passes service type into kitchen ticket data for correct grouping
- Stronger validation — service type changes are blocked on voided items and closed orders
